# Cici — Genshin Impact Companion App

Native Android application built in Java as a university course project. 
Serves as an in-game reference guide for Genshin Impact players.

## Features
- 📖 Character, weapon, and artifact guide catalog
- ❓ Interactive quiz with score tracking
- 🎵 Background music playback via Android Service
- 🔗 Useful resources section

## Tech Stack
- Java (Android SDK)
- RecyclerView with custom adapters
- Android Service (background audio)
- Intent-based multi-screen navigation
- ConstraintLayout / XML UI
